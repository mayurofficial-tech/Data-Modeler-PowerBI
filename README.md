# 📊 Data Modeler - Power BI Star Schema Project

## 📌 Description
Data Modeler is a Power BI project focused on building a normalized star schema data model for efficient data analysis and reporting.

The project demonstrates how to design a scalable and optimized data model using fact and dimension tables, ensuring high performance and accurate analysis.

## 🎯 Objective
The objective of this project is to design a robust data model using star schema architecture, define relationships between tables, and enable efficient analytical reporting.

## 📂 Data Model
The model consists of:
- Fact Tables:
  - Sales_Fact
  - Returns_Fact
- Dimension Tables:
  - Customer_Dim
  - Product_Dim
  - Region_Dim
  - Date_Dim

## 🚀 Features
- Star schema data modeling  
- Fact and dimension table design  
- One-to-many relationships  
- Optimized data filtering  
- Hierarchical data analysis  
- Clean and structured data model
